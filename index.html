API Dokumentationen, Johan Bergström

Rutter:


  • GET /users - Returnerar alla användare.

  • GET /users/{id} - Returnerar användaren med angivet ID , eller status 404 om användaren saknas.

  • POST /users - Skapa en ny användare och acceptera formaterade JSON-objekt {"firstname": "", "lastname": "", "userId": "unikt namn", "passwd": ""}. userId är obligatoriskt.

  • PUT /users/:id - Uppdaterar användaren med angivet ID och accepterar formaterade JSON-objekt {"name": "", "password": "", "email": ""}.
